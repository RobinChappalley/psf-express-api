| Méthode HTTP | Point de terminaison               | Description                                            |
| ------------ | ---------------------------------- | ------------------------------------------------------ |
| GET          | /camp/{id}                         | Récupérer un camp                                      |
| GET          | /camps                             | Récupérer la liste de tous les camps                   |
| POST         | /camp                              | Ajouter un camp                                        |
| PUT          | /camp/{id}                         | Mettre à jour un camp                                  |
| DELETE       | /camp/{id}                         | Supprimer un camp                                      |
| GET          | /camp/{id}/items                   | Récupérer tous les équipement d'un camp                |
| POST         | /camp/{id}/item                    | Ajouter un équipement pour un camp                     |
| PUT          | /camp/{id}/item                    | Mettre à jour un équipement d'un camp                  |
| DELETE       | /camp/{id}/item                    | Supprimer un équipement                                |
| GET          | /camp/{id}/trainings               | Récupérer la liste de tous les entraînements d'un camp |
| POST         | /camp/{id}/training/{id}           | Ajouter un entraînement d'un camp                      |
| PUT          | /camp/{id}/training/{id}           | Modifier un entraînement d'un camp                     |
| DELETE       | /camp/{id}/training/{id}           | Supprimer un entraînement d'un camp                    |
| GET          | /camp/{id}/training/{id}/items     | Récupérer les objets d'un entraînement                 |
| POST         | /camp/{id}/training/{id}/item/{id} | Ajouter un objet d'un entraînement                     |
| PUT          | /camp/{id}/training/{id}/item/{id} | Modifier un objet d'un entraînement                    |
| DELETE       | /camp/{id}/training/{id}/item/{id} | Supprimer un objet d'un entraînement                   |
| GET          | /hike{id}                          | Récupérer une marche                                   |
| GET          | /hikes                             | Récupérer la liste de toutes les marches               |
| POST         | /hike                              | Ajouter une marche                                     |
| PUT          | /hike/{id}                         | Mettre à jour une marche                               |
| DELETE       | /hike/{id}                         | Supprimer une marche                                   |
| GET          | /items                             | Récupérer la liste de tous les objets                  |
| PUT          | /item/{id}                         | Modifier un objet                                      |
| DELETE       | /item/{id}                         | Supprimer un objet                                     |
| POST         | /item                              | Ajouter un objet                                       |
